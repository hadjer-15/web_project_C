Tim Berners-Lee

Tim Berners-Lee is a British computer scientist best known as the inventor of the World Wide Web. He proposed the idea of a global hypertext system in 1989 and implemented the first successful communication between a Hypertext Transfer Protocol (HTTP) client and server via the internet in 1990. He also founded the World Wide Web Consortium (W3C) to oversee web standards and promote the web's growth.
